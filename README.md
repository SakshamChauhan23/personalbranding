# SocialRipple (LinkedIn)

Social Media Manager Command Center for managing multiple CXO clients at scale.

## Features
- **Onboarding**: Capture client goals and tone.
- **Profile Audit**: AI analysis of positioning and audience.
- **Content Calendar**: Generate 12-post strategies.
- **Script Writer**: Draft full posts with hooks and CTAs.
- **Admin Dashboard**: View all clients.

## Tech Stack
- **Frontend**: Next.js 14 (App Router), Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: Supabase
- **AI**: Google Gemini Pro

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Copy `.env.example` to `.env.local` and fill in your keys.
    ```bash
    cp .env.example .env.local
    ```

3.  **Database Setup**:
    Run the SQL in `supabase/schema.sql` in your Supabase dashboard.

4.  **Run the app**:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser.
